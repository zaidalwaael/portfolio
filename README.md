# Personal Portfolio Website

A modern and responsive personal portfolio website built with **HTML, CSS, and JavaScript** to showcase my background, technical skills, featured projects, education, and contact information.

This project reflects my journey as an aspiring **.NET Full Stack Developer**, with a strong focus on clean layout structure, responsive design, simple interactions, and clear presentation of my work.

## Overview

The portfolio was designed to serve as a professional personal website where visitors can:

- Learn more about me and my background
- Explore my technical skills
- View selected projects with live demos and GitHub links
- See my academic experience and collaboration highlights
- Contact me directly through a working contact form

## Features

- Responsive and clean user interface
- Simple navigation bar with mobile menu toggle
- Landing section with introduction and call-to-action buttons
- About Me section with career journey timeline
- Technical Skills section
- Featured Projects section with demo and GitHub links
- Experience and teamwork highlights
- Education section
- Contact section with Formspree integration
- Footer with dynamic project branding

## Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (ES6)**
- **Font Awesome**
- **Google Fonts**
- **Formspree** for contact form handling

## Sections Included

### 1. Hero Section

Introduces the portfolio owner with a short professional summary and quick action buttons such as viewing projects, downloading the CV, and contacting directly.

### 2. About Me

Presents an overview of academic background, interests, and development goals.

### 3. Career Journey

Shows a simple visual path from **Frontend** to **Backend** and toward **Full Stack Development**.

### 4. Skills

Highlights the main technical skills including frontend, backend, database, version control, responsive design, and UI/UX basics.

### 5. Projects

Displays featured projects with:

- Project title
- Short description
- Technologies used
- Live demo link
- GitHub repository link

### 6. Experience

Covers academic work and teamwork experience.

### 7. Education

Shows university degree information.

### 8. Contact

Includes direct links to email, GitHub, LinkedIn, and a working contact form.

## Getting Started

To run the project locally:

```bash
# Clone the repository
git clone https://github.com/zaidalwaael/portfolio.git
# Open the project folder
cd portfolio
```

Then open `index.html` in your browser.

## Deployment

This project can be deployed easily using:

- **GitHub Pages**
- **Netlify**
- **Vercel**

For GitHub Pages, make sure your main file is named `index.html` and that your asset paths are correct.
